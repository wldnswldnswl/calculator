## eval()함수

- 계산식을 계산한 결과를 반환
- ex
    - eval(2+5*6) //32
    - eval("2") //2
    - 주의
        - eval("") //undefined
        - eval() //undefined
        - eval(2*+) //와 같은 올바르지 않은 수식 ⇒ syntax error
